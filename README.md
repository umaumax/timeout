# timeout

simple timeout command by rust

## help
``` bash
timeout 0.0.1
simple timeoutcat command by rust

USAGE:
    timeout <duration> <command>...

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <duration>      duration
    <command>...    command
```

## how to use
``` bash
timeout 3s ping 127.0.0.1
```
